# Foundry Models Catalog

The **Foundry Models Catalog** is a library of AI models available in Azure AI Foundry.

It lets you:

* Browse available models
* Compare models
* Read model details
* Deploy models

## Examples of Models

* GPT-4.1
* GPT-4o
* Phi
* Llama
* Mistral
* Cohere

## Workflow

```text
Foundry Models Catalog
          │
          ▼
   Select a Model
          │
          ▼
    Deploy the Model
          │
          ▼
 Use the Deployed Model
 in your AI Application
```
